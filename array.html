Exercises

  1. Given array of ints, return maximum subarray sum (Kadane's algorithm).
  2. Given a string, find the length of the longest substring without repeating characters.
  3. Given two strings, check if one is an anagram of the other.

Solutions (brief)

Kadane (JS)

function maxSubarray(arr){
  let maxSoFar = -Infinity, cur = 0;
  for (let x of arr){
    cur = Math.max(x, cur + x);
    maxSoFar = Math.max(maxSoFar, cur);
  }
  return maxSoFar;
}

Longest substring w/o repeat (idea)

from collections import Counter
def is_anagram(a,b):
