Exercises

  1. Build a responsive two-column layout that collapses to one column on mobile (use grid or flex).
  2. Center an element both vertically and horizontally.

Hints

Use display:grid; grid-template-columns: repeat(2,1fr) and media queries to change to one column. For centering, use flexbox with align-items:center; justify-content:center;.
