Module 4: Responsive Design

Media queries, mobile-first, fluid layouts
Home

Principles

  • Mobile-first: style for small screens first, then enhance
  • Use flexible units: rem, %, vw
  • Media queries for breakpoints
  • Test across devices (emulate with browser devtools)

Example

/* mobile-first */
.container{padding:16px}
@media(min-width:768px){
  .container{padding:28px}
  .two-col{display:grid;grid-template-columns:1fr 1fr;gap:16px}
}

Exercises

  1. Create a responsive navbar that becomes a hamburger menu on small screens.
  2. Build an image gallery that adjusts columns by viewport width.

Hints

For hamburger: toggle a class to show/hide nav items; consider ARIA attributes for accessibility. For gallery: CSS grid with grid-template-columns: repeat(auto-fit, minmax(200px,1fr)).
