Why HTML matters

HTML defines the structure of web pages. Semantic HTML improves accessibility, SEO, and maintainability.

Key elements

  • <header>, <nav>, <main>, <footer>
  • <section>, <article>, <aside>
  • Forms: <form>, <input>, <label>, <button>
  • Images & media: <img> alt text, <video> controls

Example: accessible form

<form action="/subscribe">
  <label for="email">Email</label>
  <input id="email" type="email" name="email" required />
  <button type="submit">Subscribe</button>
</form>

Exercises

  1. Create a semantic blog post layout using <article>, <header>, <section>, <footer>.
  2. Build a form with validation and proper labels.

Notes & Tips

Always provide descriptive alt text for images, use headings in order (h1 → h2 ...), and prefer semantic tags over generic <div> when possible.
