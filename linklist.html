Intro

A linked list is a linear collection of nodes where each node stores data and a reference to the next (and optionally previous) node. Good when you need O(1) insert/delete at arbitrary positions given the node pointer.

Key operations

  • Traversal (O(n))
  • Insertion at head/tail/middle
  • Deletion given node
  • Reverse list (iterative & recursive)

Examples

Reverse singly linked list (JS — iterative)

function reverseList(head){
  let prev = null, cur = head;
  while(cur){
    let next = cur.next;
    cur.next = prev;
    prev = cur;
    cur = next;
  }
  return prev;
}

Detect cycle (Python — Floyd's)

def has_cycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:
            return True
    return False

Exercises

  1. Merge two sorted linked lists (return merged head).
  2. Remove N-th node from end of list in one pass.
  3. Given head, palindrome check using O(1) extra space.

Solutions (hints)

  • Merge: walk both lists and splice smaller nodes.
  • Remove N-th: use two pointers with gap n.
  • Palindrome: find middle, reverse second half, compare halves, restore if needed.
