Concepts

  • Stack: Last-In-First-Out. Use for DFS, parentheses matching.
  • Queue: First-In-First-Out. Use for BFS, scheduling.
  • Deque: double-ended queue for sliding window.

Examples

Valid parentheses (JS)

function isValid(s){
  const m = {')':'(',']':'[','}':'{'}, st = [];
  for (let ch of s){
    if ('([{'.includes(ch)) st.push(ch);
    else {
      if (st.pop() !== m[ch]) return false;
    }
  }
  return st.length===0;
}

BFS using queue (Python)

from collections import deque
def bfs(start):
    q = deque([start])
    visited = set([start])
    while q:
        node = q.popleft()
        # process node
        for nei in neighbors(node):
            if nei not in visited:
                visited.add(nei)
                q.append(nei)

Exercises

  1. Implement a min stack that returns min in O(1).
  2. Given daily temperatures, find for each day how many days to wait for a warmer temp (use stack).
  3. Implement queue with two stacks.

Solutions (brief)

Min stack: keep a stack of current minimums or store pair (value, currentMin). Two-stack queue: push to inbox, pop from outbox; transfer when outbox empty.
