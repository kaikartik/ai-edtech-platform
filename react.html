Core ideas

  • Component-based UI — small reusable pieces
  • JSX — markup inside JavaScript
  • Props (read-only) and state (mutable within component)
  • Hooks: useState, useEffect, useRef

Example: counter (React)

import React, { useState } from 'react';

export default function Counter(){
  const [count, setCount] = useState(0);
  return (
    <div>
      <h3>{count}</h3>
      <button onClick={() => setCount(c => c + 1)}>Increment</button>
    </div>
  );
}

Notes

Prefer functional components + hooks. Keep components small and focused. Lift state up when multiple children need it.
